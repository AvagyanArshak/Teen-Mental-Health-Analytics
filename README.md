# 🧠 Teen Mental Health & Social Media Analytics

## 📌 Նկարագրություն
Այս project-ը վերլուծում է 1200 դեռահասի հոգեկան առողջության տվյալները՝ կապված սոցիալական մեդիայի օգտագործման հետ։

## 📊 Dataset
- **Աղբյուր:** Kaggle — Teen Mental Health Dataset
- **Տողեր:** 1200
- **Սյունակներ:** 13

## 🔍 Եզրակացություններ

- 📱 **TikTok**-ը ամենաբարձր addiction_level-ն է առաջացնում, 
  համեմատած Instagram-ի և Both-ի հետ
- 👫 **Male**-երը մի փոքր ավելի բարձր stress ունեն քան female-երը 
  (5.47 vs 5.42)
- 😴 **Քնի ժամերը** կապ չունեն stress-ի հետ — 
  տվյալները անկանոն բաշխված են
- 🤝 **Social interaction level**-ը (high/medium/low) 
  գրեթե նույն anxiety-ն է ցույց տալիս բոլոր խմբերում
- 👥 **Gender բաշխումը** գրեթե հավասար է — 
  51.25% male, 48.75% female
